# ☕ CafeB – Full Stack Cafe Management Website

A full-stack cafe management web application designed to streamline ordering, inventory, and delivery operations with a modern and responsive interface.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup)
- 🛒 Dynamic Menu & Cart System
- 📦 Order Management System
- 🚚 Delivery Management
- 📊 Admin Panel for monitoring operations
- 📉 Stock Tracking & Inventory Control

---

## 🛠 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB (for user data, menu, orders, and stock management)
